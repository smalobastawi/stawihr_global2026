Navigate the System

The StawiHR sidebar mirrors the modules in this guide. Expand any module with the arrow to see its sub-screens.

Top navigation bar

Above the main content area, the top bar provides:

  • Menu toggle — collapse or expand the left sidebar.
  • Company selector — switch between companies (e.g. "All Companies" for super admins).
  • Financial Year — shows the active financial year (e.g. 01 Jan – 31 Dec, 2026).
  • Notifications — bell icon for system alerts.
  • User profile — your username with profile and logout options.

Permissions

You only see modules granted to your role. If a menu item is missing, ask your administrator to review Administration → Role Permissions.

Dashboard shortcuts

The dashboard provides quick-link tiles to Manage Employees, Payroll, Attendance, and Leave Management — the same modules available in the sidebar.
