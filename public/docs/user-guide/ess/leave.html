Leave & Approvals

Apply for leave, track application status, and approve team requests as a supervisor.

Apply for leave

  1. Go to Self Service → Leave.
  2. Click Apply Leave (or the equivalent action button).
  3. Select the leave type (e.g. Annual, Sick, Compassionate).
  4. Enter start and end dates. The system calculates working days based on your leave group and holidays.
  5. Provide a reason and attach supporting documents if required.
  6. Submit the application. It enters the approval workflow configured by HR.

Track your applications

Your leave list shows each application with its current status — Pending, Approved, Rejected, or Cancelled. Click an application to view approval history and comments from approvers.

Scheduled leaves

Use Self Service → Scheduled Leaves to view leave that has been planned in advance or scheduled by HR.

Approvals (supervisors)

If you are a supervisor or approver:

  1. Open Self Service → Approvals.
  2. Review pending items — leave applications, loan requests, and other workflow items assigned to you.
  3. Open each item, review details, then Approve or Reject with optional comments.
  4. Check the Submissions tab to see items you have already actioned.
Delegation If you will be away, ask HR about approval delegation so another user can action requests on your behalf.
