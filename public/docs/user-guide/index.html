Welcome to StawiHR

Your complete guide to navigating StawiHR — structured to match the application sidebar menu.

Who is this guide for? Employees using Self Service (ESS) and administrators using HR modules — Administration through Settings.

Recent features

The following capabilities were added or expanded recently:

  • Multi-currency payroll — statutory calculations in base currency with optional foreign-currency bank payments and exchange rate management.
  • Analytics hub — interactive dashboards for headcount, payroll, attendance, leave, performance, recruitment, PDP, disciplinary, training, and feedback.
  • Personal Development Plans (PDP) — employee development goals, progress tracking, and HR reports.
