Disciplinary

Record and manage disciplinary cases and actions against employees.

Categories

Disciplinary → Categories — define types of misconduct (e.g. lateness, policy violation, insubordination).

Cases

Disciplinary → Cases

  1. Open a new case and link it to an employee.
  2. Select the category and enter incident details, date, and supporting notes.
  3. Attach evidence documents if required.
  4. Track case status through investigation and resolution.

Disciplinary Actions

Disciplinary → Disciplinary Actions — record outcomes such as verbal warning, written warning, suspension, or dismissal. Actions appear on the employee's ESS → Disciplinary view (read-only).
