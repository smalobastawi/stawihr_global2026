Employee Feedback

Collect, categorize, and review employee feedback — including anonymous submissions.

Categories

Employee Feedback → Categories — define feedback types (e.g. Workplace, Management, Benefits, Suggestions).

Employee Feedbacks

Employee Feedback → Employee Feedbacks — review named feedback submitted by employees via ESS → ESS Feedback. Assign status, respond, or escalate to relevant departments.

Anonymous Box

Employee Feedback → Anonymous Box — read submissions from ESS → Anonymous Feedback where the submitter's identity is hidden. Handle sensitively and follow your organisation's whistleblowing policy.
