Notice Board

Publish company-wide announcements visible on the dashboard and in Self Service.

Notices

Notice Board → Notices

  1. Create a new notice with title, body text, and optional attachment.
  2. Set publish and expiry dates if the notice is time-limited.
  3. Publish — the notice appears on the Dashboard and under ESS → Notices.
Best practice Use notices for time-sensitive updates (office closures, policy changes). Archive expired notices to keep the board relevant.
